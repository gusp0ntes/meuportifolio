Telecompany - Faculdade

O projeto "Telecompany" foi desenvolvido como parte de um trabalho acadêmico na faculdade. O objetivo era criar um site interativo para uma empresa de telecomunicações que não apenas apresentasse informações sobre seus serviços, mas também integrasse um banco de dados para gerenciar e exibir dados dinâmicos. Durante o desenvolvimento, implementei funcionalidades como cadastro e consulta de clientes, gerenciamento de planos e serviços, e geração de relatórios. Utilizei HTML5 e CSS3 para a estrutura e o design do site, JavaScript para a interatividade do front-end, PHP para o processamento de dados e comunicação com o banco de dados, e um sistema de banco de dados SQL para armazenar e recuperar informações.

CentralGeek - Faculdade

O projeto "CentralGeek" foi desenvolvido como parte de um trabalho acadêmico na faculdade. O objetivo era criar um aplicativo móvel funcional e intuitivo utilizando o framework Ionic. Este projeto envolveu o desenvolvimento de uma aplicação para dispositivos móveis com uma interface moderna e responsiva. Utilizei o Ionic para criar uma experiência de usuário consistente em diferentes plataformas, combinando Angular e TypeScript para a lógica do aplicativo. O design foi aprimorado com SASS, e o Firebase foi integrado para gerenciar a autenticação de usuários e o armazenamento de dados em tempo real. Além disso, implementei APIs para consultar e exibir conteúdos dinâmicos no aplicativo, garantindo que as informações fossem sempre atualizadas e relevantes.
